# 2.4 Creación y uso de paquetes/librerías definidas por el usuario.   

Además de utilizar las librerias proporcionadas por el lenguaje, los programadores también pueden crear sus propios paquetes o librerias para reutilizar código en diferentes proyectos.
La creación de paquetes/librerias definidas por el usuario implica agrupar funcionalidades relacionadas en un conjunto coherente de archivos o módulos. Estos paquetes/librerias pueden incluir clases, funciones, constantes y otros elementos necesarios para llevar a cabo tareas especificas.
Una vez creados, estos paquetes/librerias pueden ser importados y utilizados en diferentes proyectos. Esto promueve la reutilización de código, evita la duplicación y facilita la gestión y actualización de las funcionalidades compartidas.
La creación y uso de paquetes/librerias definidas por el usuario permite a los programadores desarrollar código modular y escalable, facilitando el mantenimiento y la evolución de las aplicaciones a lo largo del tiempo.

Ejemplo
```python
# main.py
import flet as ft
# AQUÍ IMPORTAS TUS PROPIAS LIBRERÍAS:
import utilidades as util
from componentes import TarjetaPuntos

def main(page: ft.Page):
    # Usando el componente de la librería 'componentes.py'
    marcador_visual = TarjetaPuntos("Estudiante TAP")
    
    # Usando la dataclass de la librería 'utilidades.py'
    datos = util.DatosJuego(usuario="Ana")

    page.add(
        ft.Text("Mi App Modular", size=30),
        marcador_visual,
        ft.Text(f"Puntos actuales: {datos.puntos}")
    )
ft.app(target=main)
```
