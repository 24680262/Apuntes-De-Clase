# 2.1 Definición conceptual de componentes, paquetes / librerías.

Los componentes, paquetes y librerías son elementos fundamentales en el desarrollo de software que permiten organizar, reutilizar y distribuir código de manera eficiente.

## Componentes
Un componente es una unidad de software que encapsula funcionalidades específicas y puede ser reutilizada en diferentes aplicaciones. Los componentes pueden ser visuales, como botones, cuadros de texto, barras de progreso, o no visuales, que proporcionan funcionalidades sin interfaz gráfica. Cada componente es independiente y puede combinarse con otros para formar aplicaciones más complejas. En programación, un componente puede representarse como una clase abstracta que define posición, tamaño, capacidad de recibir eventos y de ser pintado en pantalla.

```python
import flet as ft
from dataclasses import dataclass
```

## Paquetes
Los paquetes son conjuntos de clases o módulos relacionados que se agrupan para facilitar su organización y reutilización. En Java, por ejemplo, se crea un paquete mediante la palabra reservada package seguida del nombre de la carpeta que contiene las clases. Las clases dentro del paquete deben ser públicas (public) y para utilizarlas en otras partes del programa se emplea la palabra reservada import (import NombreDelPaquete.*;). Los paquetes permiten estructurar el código según su funcionalidad, herencia o relación lógica, evitando conflictos de nombres y facilitando el mantenimiento. 
``` pip install flet ``` 

## Librerías
Una librería es un conjunto de componentes o módulos que proporcionan funcionalidades específicas listas para ser utilizadas en aplicaciones. Las librerías pueden ser dinámicas, donde el enlace con los objetos contenidos se realiza en tiempo de ejecución, o estáticas, donde el código se integra durante la compilación. Para crear librerías dinámicas, el código objeto debe ser independiente de la posición (-fPIC) y se utiliza la opción -shared para generar un fichero compatible. Las librerías permiten reutilizar código, acelerar el desarrollo y mantener consistencia en las aplicaciones.
```python
class TarjetaPerfil(ft.Container):
    # ... todo tu código aquí ...
```
