# 2.3 Creación de componentes (visuales y no visuales) definidos por el usuario

En algunos casos, puede ser necesario crear componentes personalizados para satisfacer necesidades específicas en una aplicación. Estos componentes pueden ser tanto visuales como no visuales, y se definen por el usuario mediante código.
Para crear componentes visuales definidos por el usuario, es necesario utilizar las capacidades de personalización y extensión proporcionadas por la biblioteca o framework utilizado. Esto puede implicar la creación de nuevas clases o la derivación de clases existentes para agregar funcionalidades adicionales o modificar el comportamiento de los componentes existentes.
En el caso de los componentes no visuales definidos por el usuario, se pueden crear nuevas clases o módulos que encapsulen funcionalidades especificas. Estos componentes pueden incluir lógica de negocio, algoritmos especializados u operaciones de bajo nivel.
La creación de componentes definidos por el usuario permite extender las funcionalidades de una aplicación de acuerdo con los requisitos especificos, ofreciendo mayor flexibilidad y adaptabilidad.

No Visual.
```python
from dataclasses import dataclass

@dataclass
class JugadorData:
    """Componente No Visual: Solo maneja datos"""
    nombre: str
    puntos: int = 0
    rango: str = "Novato"
```
Visual.
```python
import flet as ft

class TarjetaJugador(ft.Container):
    """Componente Visual: Define cómo se ve un Jugador en la App"""
    def __init__(self, datos: JugadorData):
        super().__init__()
        self.datos = datos  # Conectamos el componente no visual
        
        # Diseño del componente
        self.content = ft.Row([
            ft.Icon(ft.Icons.PERSON_ROUNDED, color="blue"),
            ft.Column([
                ft.Text(self.datos.nombre, weight="bold"),
                ft.Text(f"Rango: {self.datos.rango}", size=12, italic=True),
            ], spacing=0),
            ft.Text(f"{self.datos.puntos} pts", weight="bold", color="green")
        ], alignment=ft.MainAxisAlignment.SPACE_BETWEEN)
        
        self.padding = 10
        self.border = ft.border.all(1, ft.Colors.OUTLINE_VARIANT)
        self.border_radius = 8
```
Ambos unidos

def main(page: ft.Page):
    page.title = "Componentes Definidos por el Usuario"
    page.padding = 20

    # --- USO DE COMPONENTES NO VISUALES ---
    # Creamos la "data" pura

    datos_beto = JugadorData(nombre="Beto López", puntos=50)

    # --- USO DE COMPONENTES VISUALES ---
    # Creamos las tarjetas pasando los datos anteriores
    tarjeta1 = TarjetaJugador(datos_ana)
    tarjeta2 = TarjetaJugador(datos_beto)

    page.add(
        ft.Text("Ranking de Jugadores", size=24, weight="bold"),

        tarjeta1,
        tarjeta2
    )
ft.app(target=main)
