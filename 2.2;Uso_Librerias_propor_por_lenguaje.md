# 2.2 Uso de librerías proporcionadas por el lenguaje

Los lenguajes de programación modernos suelen ofrecer una amplia gama de librerias estándar que proporcionan funcionalidades comunes y útiles para el desarrollo de software. Estas librerias están incluidas en la distribución del lenguaje y se pueden utilizar directamente en el código sin necesidad de instalar o descargar nada adicional.

El uso de librerias proporcionadas por el lenguaje permite a los programadores aprovechar funcionalidades ya implementadas y probadas, lo que ahorra tiempo y esfuerzo en el desarrollo de software. Estas librerias cubren diferentes áreas, como manipulación de archivos, gestión de redes, operaciones matemáticas, procesamiento de texto, manipulación de fechas y horas, entre otros.

Para utilizar una libreria proporcionada por el lenguaje. generalmente se requiere importarla en el código fuente y luego llamar a las funciones o utilizar las clases y métodos proporcionados por esa libreria. Esto se realiza mediante declaraciones de importación especificas en el código.

Ejemplos
```python
from dataclasses import dataclass
```
```python
import warnings
warnings.filterwarnings("ignore", category=DeprecationWarning)
```
